*input

사용자에게 데이터 입력받을때 사용, 블록요소처럼 가로/세로너비와 위/아래/좌/우 여백 모두 지정가능

type 속성에는 사용자에게 입력받을 데이터의 타입을 지정


value 속성 사용시 원하는 글자 미리 지정가능


위와같이placeholder속성 사용시 입력값의 힌트적용 가능


disabled속성을 추가하면 입력요소를 비활성화 가능


input 텍스트 뿐만아니라 체크박스도 가능


labe사용시 체크박스 옆에 텍스트 선택해도 체크 가능함


강한솔 선택시 이상목해제 반대로도 가능
